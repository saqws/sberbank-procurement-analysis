# Анализ закупок группы Сбербанк (2024-2025)

## Описание проекта

Комплексный анализ закупочной деятельности группы Сбербанк на основе данных из 30 открытых источников за период 2024-2025 годов. Проект включает полный цикл работы с данными: сбор, обработку, статистический анализ, машинное обучение и визуализацию.

**Объем работы:**
- Проанализировано 150 347 закупок на сумму 2.5 трлн рублей
- Интегрировано 30 источников данных
- Реализовано 16 Python модулей (5500+ строк кода)
- Создано 15 аналитических SQL запросов
- Построено 25+ визуализаций
- Написано 1700+ строк документации

## Исследовательские гипотезы

### H1: Корреляция IT-закупок с курсом USD/RUB > 0.6
**Результат:** ПОДТВЕРЖДЕНА  
Коэффициент корреляции Пирсона составил 0.72 (p < 0.001). При росте курса доллара на 10% объем IT-закупок увеличивается на 7-8%. Выявлена критическая зависимость от импорта.

### H2: Доля монопольных закупок > 15%
**Результат:** ПОДТВЕРЖДЕНА  
Монопольные закупки (1 участник) составили 18.7%. Средняя экономия: 0.3% против 12.4% при конкуренции. Потенциал оптимизации: 42 млрд руб./год.

### H3: Сезонность строительных закупок (пик Q2-Q3)
**Результат:** ПОДТВЕРЖДЕНА  
Апрель-август: объемы выше среднего на 34-47%. Декабрь-январь: ниже на 28-35%. Коэффициент вариации 34.2%.

### H4: Ключевая ставка ЦБ влияет на строительные закупки
**Результат:** ЧАСТИЧНО ПОДТВЕРЖДЕНА  
Корреляция -0.41 (p = 0.03). Умеренная отрицательная связь. Требуется анализ с лагами 6-12 месяцев.

### H5: Топ-20 поставщиков контролируют > 30% рынка
**Результат:** ПОДТВЕРЖДЕНА  
Топ-20: 34.2% рынка (851 млрд руб.). Топ-5: 22.8% (567 млрд). Индекс HHI = 0.18. Выявлен риск концентрации.

## Ключевые результаты

### Динамика
Рост объема закупок на 18% в 2025 году несмотря на санкции. Провал в марте-апреле 2024 (-23%) с восстановлением к августу. Система показала высокую адаптивность.

### Аномалии (247 случаев, 0.16%)
- **89 закупок** с экономией > 50% (среднее 64.3%) - возможное завышение НМЦК
- **67 закупок** с перерасходом > 20% (среднее 28.7%) - изменение условий контрактов
- **91 закупка** с аномальными победителями (доля побед > 30%)

### Структура по категориям
- IT: 28.3% (наибольшая волатильность)
- Строительство: 23.7%
- Консалтинг: 14.2%
- Оборудование: 12.8%
- Прочее: 21.0%

### Конкурентность
Среднее участников на процедуру: 2.4
- 1 участник: 18.7%
- 2 участника: 31.2%
- 3 участника: 24.8%
- 4-5 участников: 17.3%
- 6+ участников: 8.0%

## Рекомендации

### 1. Хеджирование валютных рисков
Внедрение форвардов и опционов для IT-закупок. Потенциальная экономия: 3-5% (22-34 млн руб./год).

### 2. Повышение конкуренции
Снижение доли монополий с 18.7% до 12% за 18 месяцев. Дополнительная экономия: 35-42 млрд руб./год.

### 3. Диверсификация поставщиков
Снижение доли топ-20 с 34% до 25% за 24 месяца. Минимизация рисков зависимости.

### 4. Расследование аномалий
Детальная проверка 247 случаев службой внутреннего контроля. Приоритет: избыточная экономия и перерасход.

### 5. Система мониторинга
Внедрение real-time аналитики с автоматическим выявлением отклонений.


## Методология

### Сбор данных
- Асинхронные запросы с rate limiting
- Retry logic с exponential backoff
- Автоматическое обезличивание ПД (regex + NER)
- Детальное логирование процесса

### Обработка
- Удаление дубликатов (exact, fuzzy, priority-based)
- Валидация с расчетом quality scores
- Нормализация форматов
- Enrichment из внешних источников

### Анализ
- Descriptive statistics
- Hypothesis testing (t-tests, ANOVA)
- Correlation analysis (Pearson, Spearman, lag analysis)
- Anomaly detection (IQR, Z-score, Isolation Forest)
- Time series decomposition (STL)
- LLM-powered insights

### Визуализация
- Интерактивные графики (Plotly)
- Статистические визуализации (Seaborn)
- Временные ряды, распределения, корреляции
- Network graphs, treemaps, heatmaps
